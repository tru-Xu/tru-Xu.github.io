Analyse for convergence condition: result - target > target 。
I don’t know how to find this condition, but I can use mathematical induction to prove this conclusion.

1. assumption

when x^n < target < x^(n+1).
we asume x^n + a = target;
x^(n+1) - b = target;

we use f(target) represent express number of target.

then we define

left = f( target) .

right = f(b) .

we can deduce the conclusion: if target + b = x^(n+1), target < b then left < right.

how to prove this?

2. mathematical induction

2.1 when n=0,

if n = 0, x^0 < target < x^1, we can get the variable left and right:
left = f(target) = 2target -1;
right = f(b) = 2(x - target) - 1;

if x - target > target

then 2(x - target) - 1 > 2target - 1

that is to say: right > left;

2.2 when n = m - 1, (m > 1)

we asume that: when n = m-1, x^(m - 1) < target < x^m, if x^m - target > target, we can get the conclusion right > left.

that is to say

f(b) > f(target), (b + target = x^m) (1 -1)

2.3 when n = m

how to prove the conclusion when n = m ?

when x^m < target < x^(m+1) (1)

and x^(m+1) - target > target. (2)

we assume that

x^m + a = target, x^(m + 1) - b = target (3)

x^m + a can be replaced by c*x^m + c0 , (c >= 1, 0 <= c0 < x^m) (4)

b can be replaced by d*x^m + d0, (d >= 1, 0 <= d0 < x^m) (5)

from (3)(4)(5), we can get :

cx^m + c0 + dx^m + d0 = x^m + a + b = x^(m + 1). (6)

from (6) we can get

c0 + d0 = 0 (6-1)

or

c0 + d0 = x^m. (6-2)

  • if c0 + d0 = 0, then c0 = 0, d0 = 0, according to (2) we can know

b > x^m + a , that means dx^m > c x^m, d > c. then we can prove right > left.

  • if c0 + d0 = x^m.

    • if c == d

      according to (2) we can know

      b > x^m + a , that means dx^m + d0 > cx^m + c0, then d0 > c0.

      c0 + d0 = x^m, c0 < d0 , according to (1-1) wo can know f(d0) > f(c0) => right > left.

    • if c < d.

      i. dx^m + d0 = (d + 1)x^m - c0

      ​ right = f(b) = f(dx^m + d0) = f((d + 1)x^m - c0)

      ​ left = f(target) =f (cx^m + c0)*

      ii. cx^m + c0 = (c+1)x^m - d0*

      ​ right = f(b) = f(dx^m + d0)*

      ​ left = f(target) =f (cx^m + c0)=f((c+1)x^m - d0)*

      we can see right > left.
